🚀 About the Project

WipeSure is a project built for Smart India Hackathon (SIH) with the goal of improving hygiene standards in public and private washrooms.

The idea is simple — many times washrooms are either not cleaned on time or lack basic supplies like tissues. WipeSure helps in solving this problem by providing a smart monitoring system that ensures cleanliness and availability of essential items.

This project focuses on making hygiene reliable, trackable, and efficient.

💡 Problem We Are Solving

In many places like colleges, malls, offices, and public areas:

Washrooms are not cleaned regularly
Supplies like tissues run out without notice
No proper system exists to track maintenance

This leads to poor hygiene and inconvenience.

🛠️ Our Solution

WipeSure introduces a system that:

Monitors usage and supply levels
Alerts staff when refill or cleaning is needed
Maintains a proper hygiene record

This helps management take action on time instead of reacting late.

⚙️ Features
📊 Smart monitoring system
🔔 Real-time alerts for maintenance
📉 Usage tracking
🧾 Data logging for analysis
🧼 Improves overall hygiene standards
